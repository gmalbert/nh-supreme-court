# Granite State Appeals

Granite State Appeals is a public-facing site for exploring the work of the New Hampshire Supreme Court.
It brings together the court's decisions, orders, justices, and case topics in one place so people can
follow what the court is doing and look for patterns over time.

The site is built for everyone who wants a clearer view of the court's work, including members of the
public, journalists, students, attorneys, researchers, and civic groups.

## What You Can Find

### Court Decisions
Read the court's published decisions, see who wrote them, and view the vote breakdown when available.

### Court Orders
Explore shorter orders that help show how the court manages its docket and handles everyday matters.

### Justices
Look up individual justices to see their role on the court, their writing history, and their voting patterns.

### Topics
Browse cases by broad subject area such as criminal law, civil disputes, family matters, and other common themes.

### Trends and Analysis
See how the court's work changes across years, how long decisions take, which topics appear most often,
and how often justices agree or disagree with one another.

### Trial Court Results
Review outcomes from lower-court cases when that information is available, along with links back to the
relevant Supreme Court material.

## How To Use The Site

Start with the page that best matches what you want to know:

- Use **Opinions** if you want to read a specific court decision.
- Use **Case Orders** if you want shorter orders and docket activity.
- Use **Justices** if you want to focus on one justice's work.
- Use **Analysis** if you want charts and broader trends.
- Use **Topics** if you want to explore cases by subject.
- Use **Trial Courts** if you want to follow lower-court outcomes.

Most pages include filters, tables, and links that make it easy to move between cases, justices, and topics.

## What This Site Is For

This site is meant to make court information easier to understand and easier to browse.
It is useful for:

- reading decisions in context
- comparing how justices approach different cases
- tracking changes in the court's work over time
- finding examples for reporting, teaching, and public discussion

## A Few Things To Know

- Court materials sometimes use formal legal language.
- Some cases are easier to classify than others.
- Names, titles, and court membership can change over time.
- If you are using the site for anything important, it is always a good idea to check the original court record.

## Important Note

Granite State Appeals is provided for general information and research.
It is not legal advice and is not affiliated with the New Hampshire Judicial Branch or any government entity.
